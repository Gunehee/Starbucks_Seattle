What it does

Built for real-world exploration

Every feature was designed to help users quickly find the right Starbucks — whether they need a drive-thru, a quiet in-store spot, or the nearest open location.

🔀

Service Type Filtering

Toggle between All Stores, In-Store Only, Drive-Thru Only, and In-Store & Drive-Thru. The map and list update instantly.

🔍

Real-Time Search

Type a store name, street, or neighborhood to filter both the map markers and the sidebar list simultaneously as you type.

📋

Alphabetical Store List

A scrollable sidebar lists every visible store alphabetically. Click any entry to fly the map directly to that location.

📍

Store Detail Panel

Selecting a store opens a slide-in panel with its full address, phone number, hours of operation, and service type badge.

🗺️

Fly-To Animation

The map smoothly pans and zooms to any selected store, keeping the user oriented within Seattle's geography.

↗️

Google Maps Directions

Every store detail panel includes a one-click "Get Directions" button that opens Google Maps with the destination pre-filled.

Data & Methodology

From raw CSV to interactive GeoJSON

The source data is the Starbucks Locations Worldwide 2021 dataset from Kaggle, cleaned and filtered to Seattle only.

1

Filter — Isolated rows where city = Seattle and country_code = US

2

Clean — Removed 5 irrelevant columns and dropped all rows with null coordinate values

3

Normalize — Converted ZIP+4 codes (e.g., 981162812) to standard 5-digit ZIP (98116)

4

Segment — Exported four GeoJSON files split by service type for the layer filter system

FileDescriptionCount
Starbucks_Seattle.geojsonAll locations102

All GeoJSON data is embedded inline within map.html — no network requests at runtime.

Built with

Open-source, zero dependency

No build tools. No framework. No API key. The entire application runs from a single HTML file in any modern browser.

🗺️

Leaflet.js v1.9.4

Interactive map rendering, marker management, and popup system

🌐

OpenStreetMap

Free base map tile layer — no API key, no CORS restrictions

📦

GeoJSON

Store location data with name, address, phone, and hours embedded inline

⚡

Vanilla JS ES6+

All filtering, search, and state management — no framework dependency

🎨

CSS3

Custom properties, transitions, and responsive layout — no preprocessor

↗️

Google Maps URLs

Coordinate-based directions deep link — no API key needed
